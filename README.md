## üöÄ How to Use These Slides with ClassPoint

### Prerequisites
- Microsoft PowerPoint (2016 or later)
- ClassPoint add-in installed
- ClassPoint account (free or premium)

### Setup Instructions

1. **Install ClassPoint**
   ```
   - Open PowerPoint
   - Go to Insert ‚Üí Get Add-ins
   - Search for "ClassPoint" and install
   - Sign in/create ClassPoint account
   ```

2. **Enable Interactive Features**
   ```
   - Open the presentation
   - Click ClassPoint tab in PowerPoint ribbon
   - Ensure "Presentation Mode" is active
   - Test quiz slides in Slide Show mode
   ```

3. **Running the Training Session**
   ```
   - Start Slide Show
   - Participants join via classpoint.app
   - Enter class code displayed on screen
   - Interactive buttons will appear during quiz slides
   - Monitor responses in real-time dashboard
   ```

### For Participants
1. Navigate to **classpoint.app**
2. Enter the class code displayed on screen
3. No account required for participants
4. Use any device with internet connection
